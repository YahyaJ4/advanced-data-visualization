# Linked Views, Brushing

## Overview
This project aims to build an interactive visualization for the well known Iris Dataset by designing linked views that enable interaction through selection and brushing.

## File Structure
- `index.html`: HTML file that contains the structure for displaying the the visual.
- `a06.js`: This file contains the JavaScript code responsible for implementing the interactive visualization features, including linked views and interactive brushing, for the Iris Dataset.
- `d3.js`: Holds the d3 library functionalities.
- `iris.js`: JavaScript file that contains the data used to generate the interactive visualizations.  

## How to Use
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. The scatterplots will be displayed based on the provided dataset.

NOTE: I used chrome as my browser (you can drag and drop the file)

If using VSCode:

You can open the program with "Open with live server".
