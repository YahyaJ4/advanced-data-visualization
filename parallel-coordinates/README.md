# Parallel Coordinates

## Overview
This project aims to build an interactive visualization that uses a parallel coordinates display to investigate the Palmer Penguins Dataset.

## File Structure
- `index.html`: HTML file that contains the structure for displaying the the visual.
- `a07.js`: JavaScript file that contains the code for generating a parallel coordinates plot from penguin dataset with two interactions features (label click, and axis brushing).
- `d3.js`: Holds the d3 library functionalities.
- `penguins.js`: JavaScript file that contains the penguins dataset, which include 4 quantitative attributes and 3 categorical attributes.

## How to Use
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. The scatterplots will be displayed based on the provided dataset.

NOTE: I used chrome as my browser (you can drag and drop the file)

If using VSCode:

You can open the program with "Open with live server".
